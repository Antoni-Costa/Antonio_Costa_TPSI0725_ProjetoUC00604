# Projeto Condomínio Digital

## Propósito
Este projeto é uma plataforma para moradores de um condomínio, permitindo visualização de serviços e contacto direto com a administração.

## Estrutura do Projeto
- /CSS/style.css: Estilização e responsividade.
- /JS/javascript.js: Validação de formulários e interatividade.
- index.html: Estrutura principal do Condomínio Digital.

## Tecnologias Utilizadas
- HTML5: Estruturação de conteúdo.
- CSS3: Design responsivo e animações hover.
- JavaScript: Validação de formulários e menu interativo.

## Como Executar
1. Faça o download dos ficheiros.
2. Abra o ficheiro `index.html` em qualquer navegador (Chrome, Firefox, Edge).
